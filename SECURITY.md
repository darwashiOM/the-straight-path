# Security Policy

Thank you for helping keep **The Straight Path** and its readers safe.

## Supported versions

Only the `main` branch deployed at https://the-straight-path-tsp.web.app is
actively maintained. Please ensure reports reference the current live site.

## Reporting a vulnerability

If you discover a security vulnerability, **please do not open a public
GitHub issue.** Instead, disclose it privately so we can investigate and
release a fix before details become public.

- **Email:** security@thestraightpath.app (or contact the maintainer,
  [@darwashiOM](https://github.com/darwashiOM), through GitHub)
- **Subject line:** `[security] <brief description>`

Please include:

1. A description of the issue and its impact.
2. Steps to reproduce (proof of concept, affected URL, request payload, etc.).
3. Your environment (browser, OS, date/time of testing).
4. Whether you would like credit in the fix notes.

## Our commitment

- We will acknowledge receipt within **3 business days**.
- We will provide an initial assessment within **7 business days**.
- We will keep you informed as we work on a fix, and credit you in the
  release notes if you wish.
- We will not pursue legal action against researchers who act in good
  faith, follow this policy, and avoid privacy violations or service
  disruption.

## Scope

In scope:

- The production site at `the-straight-path-tsp.web.app` and any custom
  domains we point at it.
- The code in this repository (web app, Cloud Functions, Firebase rules).

Out of scope:

- Third-party services (Firebase platform itself, font CDNs, etc.) — please
  report those to the respective vendor.
- Social engineering of maintainers or contributors.

## Thanks

This project exists to help people. We're grateful for any time you spend
making it safer.
